de.schenk.jrtrace.helperlib includes

ASM 5.0.4 repackaged to de.schenk.objectweb.asm.

Note: the package de.schenk.objectweb.addons was added for JRTrace and is not part of the ASM project.


Note: to deploy a new version of ASM the below will do:

- download the latest source
- rearrange the directory structure from org... to de.schenk... in the filesystem
- perform a full text search/replace on all files from org.objectweb.asm to de.schenk.objectweb.asm
- replace
